# trabalho_mvc_exercicio2

# Sistema de Biblioteca Escolar

## Classes usadas

- Livro
- Aluno
- Emprestimo

## Regras do sistema

- Não pode cadastrar livro sem título
- Não pode cadastrar livro com quantidade negativa
- Não pode emprestar livro se não tiver disponível
- Deve ser possível devolver o livro
- O sistema precisa mostrar quais livros estão emprestados
- Também deve mostrar quais alunos ainda não devolveram

## Relações

- Um aluno pode pegar vários livros emprestados
- Um livro pode ser emprestado várias vezes
